# browser-chatgptapi

将网页版ChatGPT做成API，

**如何使用**：[如何使用？](wiki/如何使用.md)


**推荐魔法网络**：[点此链接有月1元优惠套餐](https://xx025.github.io/773ycd9u.html)


### 实现思路：

一个web服务器，chatgpt网页利用注入脚本与服务器通信，接收查询语句并返回,利用sowebscoekt 协议与服务器进行通信。
